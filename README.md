# way-to-multiply

## Index

 - Ethiopian Multiplication

## Process

- 두 수 a,b가 있다고 할 때
  
1. a의 값이 홀수라면 b를 더한다.
2. a를 2로 나눈 값을 a에 저장하고 b에 2를 곱한 값을 b에 저장한다.
3. 1과 2의 과정을 a가 1일 때까지 반복한다.
4. 최종 곱셈결과를 출력한다.
